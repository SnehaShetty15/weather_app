# SETUP INSTRUCTIONS - Smart Weather App

## Follow these steps to get your app running:

### 1. Get Your FREE OpenWeatherMap API Key
   - Visit: https://openweathermap.org/api
   - Click "Sign Up" and create a free account
   - After login, go to: https://home.openweathermap.org/api_keys
   - Copy your API key

### 2. Configure This File
   - Rename this file from SETUP_INSTRUCTIONS.txt to .env
   - Replace "your_api_key_here" below with your actual API key
   - Save the file

### 3. Install Dependencies
   Open terminal/command prompt in this folder and run:
   pip install -r requirements.txt

### 4. Run the App
   python app.py

### 5. Open Browser
   Go to: http://localhost:5000

---

# OpenWeatherMap API Key
